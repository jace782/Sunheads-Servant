About the Bot

Sunhead's Servant is a powerful Discord moderation and utility bot designed to keep your server clean, organized, and connected. With advanced role management, Roblox game linking, and message proxy commands, it gives server owners full control.

Core Features

  • Role Management: Create, delete, assign, and remove roles using simple ! commands.
  • Roblox Game Linking: Use !link to connect your Roblox game to the bot via a custom URL.
  • Proxy Messaging: Send messages as the bot using !write #channel message.
  • Utility Tools: Helpful commands for server owners and admins.

Command Style

All commands use the classic prefix format: 

!link 
!write #channel your message here
!createrole 
!giverole @user
