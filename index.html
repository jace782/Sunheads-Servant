About the Bot

Sunhead's Servant is a sweet but powerful Discord moderation and utility bot. It keeps your server safe, organized, and even connects to your Roblox game.

Core Features

  • Role Management: Create, delete, assign, and remove roles with ! commands.
  • Roblox Game Linking: Use !link to connect your game to the bot.
  • Proxy Messaging: Send messages as the bot using !write #channel message.
  • Utility Tools: Helpful commands for server owners.

Command Style

!link <game-url>
!write #channel your message here
!createrole <name>
!giverole @user <role>
